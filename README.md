# SharkyBot
SharkBotのMisskeyバージョン

# 使い方
```
botをメンション /コマンド名
```
とノートを投稿すれば使用できます。

# コマンド一覧

### 基本コマンド
```
コマンド名 | 説明
help | このサイトのURLを出します。
user | Misskeyのユーザーを検索します。
ping | Pong!と返します。
now | 現在時刻を返します。
wiki | Wikiを検索します。
server | Misskeyサーバー情報を取得します。
news | トップニュースを取得します。
```

### ネタコマンド
```
コマンド名 | 説明
good | いいねリアクションを付けます。
miq | 返信したメッセージを迷言化します。
5000 | 5000兆円ほしい！
roll | さいころです
omikuji | おみくじを引きます。
```

### 経済コマンド
```
コマンド名 | 説明
work | 仮想のお金を稼ぎます。
bal | 現在残高を見ます。
```
