# SharkyBot
SharkBotのMisskeyバージョン

# 使い方
```
botをメンション /コマンド名
```
とノートを投稿すれば使用できます。

# コマンド一覧

### 基本コマンド
```
コマンド名 | 説明
help | このサイトのURLを出します。
good | いいねリアクションを付けます。
user | Misskeyのユーザーを検索します。
miq | 返信したメッセージを迷言化します。
ping | Pong!と返します。
now | 現在時刻を返します。
roll | さいころです
wiki | Wikiを検索します。
```

### 経済コマンド
```
コマンド名 | 説明
work | 仮想のお金を稼ぎます。
bal | 現在残高を見ます。
```
